<h1 align="center"> 🏦 A CLI BANKING APP 🏦</h1>

A console-based banking application built in Java for teller use.
Focuses on secure customer handling, account management, and reliable transaction processing.

## Overview
Console-based banking system for tellers.  
Focus on customer authentication, account management, and secure transaction handling.

## Core Requirements
- Authenticate customers before any operation
- Support account types:
    - Personal
    - ISA
    - Business
- Perform operations:
    - Deposit, withdraw, transfer
    - Balance checks
- Generate unique 8-digit account numbers automatically
- Assign sort codes based on account type
- Maintain transaction history

## Technical Expectations
- Strong OOP design (inheritance, polymorphism, separation of concerns)
- Input validation and error handling (no crashes)
- Persistent data storage (JSON/CSV/serialization)
- Logging system with timestamps

## Key Rules
- Personal: min £1 balance, overdraft allowed
- ISA: one per customer, 2.75% interest
- Business: £120 annual fee, one per customer

## Deliverables
- Java source code
- Assumptions document
- User guide
- Test evidence
- UML diagrams
